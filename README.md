## Run on your Local Machine

Clone the project

```bash
  git clone https://github.com/[your-username]/SharkSaw-Puzzle.git
```

Go to the project directory

```bash
  cd SharkSaw-Puzzle
```

Install dependencies

npm

```bash
  npm install
```
yarn

```bash
  yarn install
```

Start the server

npm

```bash
  npm run start
```
yarn

```bash
  yarn run dev
```

It will start the development server at `localhost:3000`
